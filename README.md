# 🔢 Pattern Generator & Number Analyzer

## 📌 Description

This Python project is a menu-driven program that allows users to generate patterns and analyze numbers.

The program provides options to:

* Generate star patterns
* Check numbers as even or odd
* Calculate the sum of a given range

---

## 🚀 Features

* Menu-driven program
* Pattern generation using loops
* Even/odd number checking
* Sum calculation of a range
* Error handling using try-except

---

## 🛠️ Tech Used

* Python (Basics)

---

## ▶️ How to Run

1. Install Python
2. Run the file:

python "logic box.py"
